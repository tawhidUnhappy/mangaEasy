# Bundled lyric font

`edosz.ttf` is the unmodified **Edo SZ Regular** font by Vic Fieger. It is the
default display face for Song Video lyric renders.

- Source: <https://www.dafont.com/edo-sz.font>
- Author terms: <https://vicfieger.com/~font/faq.html>
- SHA-256: `BC67CF1C852C6D4FFBB7BC8FB4CD702D293EE49BDA3DE12C94E3635D80A4D55B`
- File size: 48,820 bytes

The author's terms permit use for personal, commercial, or charitable design
work and permit including the font with software; do not sell the font itself.
The font has a limited Latin character set. Song manifests may select another
properly licensed font for languages or glyphs it does not cover.
